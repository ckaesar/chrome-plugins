# chrome-plugins
想要的插件自己实现

## 🛠️ 工具列表 (Tools List)

### 1. 树叶倒计时 (Leaf Countdown Timer)
**目录**: `./countdown-timer-extWension`

**核心功能**:
- ⏱️ **自定义倒计时**: 支持分/秒设置，界面简洁清新。
- 🍃 **到期提醒**: 倒计时结束后通过浏览器系统通知弹窗提醒（标题带 🍃 标志）。
- 💾 **自动记忆**: 自动保存上次设置的时间，方便重复使用。
- ⚙️ **后台运行**: 基于 Chrome Alarms API，即使关闭插件弹窗也能准确计时。
